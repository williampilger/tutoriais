# Comando de Terminal do VMWare

É possível controlar o vmware pelo terminal e por SSH.

| Comando | Descrição |
| --- | --- |
| `vmrun list` | Listar as VMs **rodando** |
| `vmrun start /caminho/para/a/vm.vmx` | Iniciar uma VM (isso só funciona se você estiver em uma seção COM DISPLAY |
| `vmrun start /caminho/para/a/vm.vmx nogui` | A opção `nogui` permite a execução da VM sem usar a interface gráfica. |
