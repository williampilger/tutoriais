# Reiniciar Interface Gráfica (Encerrando as sessões todas)

Muitas vezes, quando a interface gráfica trava totalmente e não adianta parar um processo em específico, pode ser útil reiniciar totalmente a interface.

Acesse o modo terminal com `Ctrl`+`Alt`+`F3` (ou `FX` que for).



#### Ubuntu (Gnome)

> sudo systemctl restart gdm


#### Xubuntu (ou derivados/semelhantes)

> sudo systemctl restart lightdm
