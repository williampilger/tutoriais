# Atalhos de Teclado do Google Gmail

Veja o [tutorial completo](https://support.google.com/mail/answer/6594?hl=pt-BR#zippy=%2Ca%C3%A7%C3%B5es) na página do google.


## Ações

| Ação | Atalho |
| --- | --- |
| **Mover foco para a barra de ferramentas** | `,` |
| **Selecionar uma conversa** | `x` |
| **Alternar estrela / opções de super estrelas** | `s` |
| **Arquivar** | `e` |
| **Ignorar conversa** | `m` |
| **Denunciar como spam** | `!` |
| **Excluir** | `#` |
| **Responder** | `r` |
| **Responder em uma nova janela** | `Shift + r` |
| **Responder a todos** | `a` |
| **Responder a todos em uma nova janela** | `Shift + a` |
| **Encaminhar** | `f` |
| **Encaminhar em uma nova janela** | `Shift + f` |
| **Atualizar conversa** | `Shift + n` |
| **Arquivar conversa e ir para a anterior/próxima** | `]` ou `[` |
| **Desfazer última ação** | `z` |
| **Marcar como lida** | `Shift + i` |
| **Marcar como não lida** | `Shift + u` |
| **Marcar como não lida a partir da selecionada** | `_` |
| **Marcar como importante** | `+` ou `=` |
| **Marcar como não importante** | `-` |
| **Adiar (Snooze)** | `b` |
| **Expandir toda a conversa** | `;` |
| **Recolher toda a conversa** | `:` |
| **Adicionar às Tarefas (Google Tasks)** | `Shift + t` |
