# Comparando Branches no Github Online

For example, this URL uses the shortened seven-character SHA codes to compare commits `f75c570` and `3391dcc`: 
> https://github.com/**ORGANIZAÇÃO**/**REPOSITÓRIO**/compare/f75c570...3391dcc`.

Ou, para comparar duas branches, `main` e `exemplo`:
> https://github.com/**ORGANIZAÇÃO**/**REPOSITÓRIO**/compare/main...exemplo`.
