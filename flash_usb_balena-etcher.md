# Gravando uma imagem ISO em um pendrive utilizando o Balena Etcher

Baixe o `Balena Etcher` para o seu sistema, disponível no [site oficial](https://www.balena.io/etcher/?).

E, depois, para gravar a imagem ISO em um pendrive, siga os passos abaixo:
1. Execute o Balena Etcher;
2. Selecione **Flash from file** e navegue até a ISO que você deseja gravar;
3. Insira e selecione o cartão SD que deseja utilizar (aconselhamos o uso de SD classe 10);
4. Clique em **Flash** e digite sua senha de administrador, caso necessário.
   
Aguarde o processo terminar, e depois é só usar o pendrive para instalar o sistema operacional que você gravou nele.
